# 크리스마스 프로모션

## 프로젝트

식당에 방문할 날짜와 메뉴를 입력하면
주문 메뉴, 할인 전 총주문 금액, 증정 메뉴, 혜택 내역,
총혜택 금액, 할인 후 예상 결제 금액, 12월 이벤트 배지
내용을 츌력한다.
